🧹 Web Quản Lý Thuê Người Giúp Việc
📌 Giới thiệu
Đây là dự án xây dựng website quản lý dịch vụ thuê người giúp việc, nhằm kết nối giữa khách hàng và người cung cấp dịch vụ.
Hệ thống hỗ trợ người dùng tìm kiếm và đặt lịch thuê một cách thuận tiện.

🚀 Công nghệ sử dụng
Backend: C#, ASP.NET Core MVC
Frontend: HTML, CSS, JavaScript, Bootstrap
Database: SQL Server
ORM: Entity Framework Core
